- _Node_: version >= 16.16
- _NPM_: version >= 8.11
- _NestJs_: >= 9.0.0
- _Docker_: version >= 20.10
- _Docker Compose_: version >= 1.29

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
